**File structure (top to bottom):**

1. **Header comment:**
```javascript
// .claude/hooks/pipeline-observe.js
// Captures Claude Code tool events to per-project obs.jsonl files.
// Registered in .claude/settings.json for: PreToolUse, PostToolUse,
// PostToolUseFailure, PermissionDenied, SubagentStop
// Run: node pipeline-observe.js <event_type>
```

2. **Imports and constants:**
```javascript
'use strict';
const fs = require('fs');
const path = require('path');

const HOOK_EVENT = process.argv[2] || 'unknown';
const PROJECT_ROOT = process.env.CLAUDE_PROJECT_DIR || process.cwd();
const OBS_BASE = path.join(PROJECT_ROOT, '.claude', 'logs', 'observations');
const MAX_SIZE = 10 * 1024 * 1024; // 10MB rotation threshold
const PURGE_DAYS = 30;
const PURGE_MS = PURGE_DAYS * 24 * 60 * 60 * 1000;
```

3. **Truncation caps per D-09:**
```javascript
const TRUNCATION_CAPS = {
  Read:  { input: 1024, output: 1024 },
  Grep:  { input: 1024, output: 1024 },
  Glob:  { input: 1024, output: 1024 },
  Bash:  { input: 5120, output: 5120 },
  Write: { input: 5120, output: 5120 },
  Edit:  { input: 5120, output: 5120 },
  Agent: { input: 2048, output: 5120 },
};
const DEFAULT_CAP = { input: 2048, output: 2048 };
```

4. **Stdin buffering and main entry (per check-memory-limit.js pattern):**
- Buffer stdin, parse JSON, call `main(data)` inside try/catch
- On any error: `process.stderr.write('[pipeline-observe] ' + err.message + '\n')`
- Always `process.exit(0)` unconditionally at end

5. **`main(data)` function:**
- Call `detectProject(data)` to get project slug
- Construct `projectDir = path.join(OBS_BASE, project)`
- `fs.mkdirSync(projectDir, { recursive: true })`
- Construct `obsFile = path.join(projectDir, 'obs.jsonl')`
- Call `rotateIfNeeded(obsFile, path.join(projectDir, 'obs.archive'))`
- Call `purgeOldArchives(projectDir)` (throttled by .last-purge marker)
- Switch on HOOK_EVENT: 'tool_pre', 'tool_post', 'tool_fail', 'permission_denied' -> `handleToolEvent(data, obsFile)`
- Case 'subagent_stop' -> placeholder comment: `// SubagentStop handler added in Plan 02`

6. **`detectProject(data)` function:**
- Build sources array: `[data.cwd || '', JSON.stringify(data.tool_input || {}), data.prompt || '', data.agent_transcript_path || '']`
- Match regex: `/projects[/\\]([a-z0-9][a-z0-9-]*)/i`
- Return first match lowercased, or `'_channel'` as default

7. **`rotateIfNeeded(obsFile, archiveDir)` function:**
- Try `fs.statSync(obsFile).size` -- if >= MAX_SIZE:
  - `fs.mkdirSync(archiveDir, { recursive: true })`
  - Build archive filename: `obs-${ts}-${process.pid}.jsonl` where `ts = new Date().toISOString().replace(/[:.]/g, '-')`
  - `fs.renameSync(obsFile, path.join(archiveDir, archiveName))`
- Catch ENOENT (file doesn't exist): silently continue
- Catch other errors: log to stderr, continue

8. **`purgeOldArchives(projectDir)` function:**
- Check `.last-purge` marker: `path.join(projectDir, '.last-purge')`
- If marker exists and mtime < 24 hours ago: skip (throttle to once per day)
- Otherwise: scan `path.join(projectDir, 'obs.archive')` directory
- For each `.jsonl` file: check `fs.statSync(f).mtimeMs` -- if older than PURGE_MS, `fs.unlinkSync(f)`
- Touch marker file: `fs.writeFileSync(purgeMarker, '', 'utf8')`
- Wrap entire function in try/catch (purge failure must never block)

9. **`handleToolEvent(data, obsFile)` function:**
- Get tool name: `const tool = data.tool_name || data.tool || 'unknown'`
- Get caps: `const caps = TRUNCATION_CAPS[tool] || DEFAULT_CAP`
- Build event object (per D-08 lean identity):
  ```javascript
  const event = {
    ts: new Date().toISOString().replace(/[:.]/g, '-'),
    epoch_ms: Date.now(),
    event: HOOK_EVENT,        // 'tool_pre' | 'tool_post' | 'tool_fail' | 'permission_denied'
    session_id: data.session_id || '',
    agent_id: data.agent_id || '',   // empty string = main conversation (per D-08)
    tool: tool,
    tool_use_id: data.tool_use_id || '',
    project: detectProject(data),
  };
  ```
- Add conditional fields based on HOOK_EVENT:
  - `tool_pre`: `event.input = truncate(JSON.stringify(data.tool_input || {}), caps.input)`
  - `tool_post`: `event.output = truncate(JSON.stringify(data.tool_response || data.tool_output || data.output || ''), caps.output)`
  - `tool_fail`: `event.output = truncate(JSON.stringify(data.tool_response || data.tool_output || ''), caps.output)` AND `event.error = truncate(String(data.error || ''), 2048)`
  - `permission_denied`: `event.input = truncate(JSON.stringify(data.tool_input || {}), caps.input)` AND `event.reason = truncate(String(data.reason || ''), 1024)`
- Call `appendEvent(obsFile, event)`

10. **`truncate(str, maxBytes)` utility:**
```javascript
function truncate(str, maxBytes) {
  if (!str) return '';
  if (typeof str !== 'string') str = String(str);
  return str.length > maxBytes ? str.slice(0, maxBytes) : str;
}
```

11. **`appendEvent(obsFile, eventObj)` function (CAPT-04):**
```javascript
function appendEvent(filePath, eventObj) {
  const line = JSON.stringify(eventObj) + '\n';
  fs.appendFileSync(filePath, line, 'utf8');
}
```
Single write syscall. Atomic on NTFS for buffers under 1MB.

- Per D-03: No legacy filter layers (no global disable, no hook profile, no cooperative skip, no agent_id gate, no path exclusions)
- Per D-04: No secret scrubbing
- Per D-07: Capture everything (all tool calls, both main and subagent)
- Per D-08: `agent_id` field only (empty string for main, populated for subagent)
- Per CAPT-07: All paths constructed with `path.join()` / `path.resolve()` -- never string concatenation
- Hook MUST exit 0 always, MUST never throw uncaught, MUST never write to stdout (only stderr for errors)
