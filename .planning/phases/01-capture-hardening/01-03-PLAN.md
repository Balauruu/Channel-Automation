3. **Helper functions:**

`makeTmpProject()` - creates temp directory with `.claude/logs/observations` structure, returns path. Use `fs.mkdtempSync(path.join(os.tmpdir(), 'obs-test-'))`.

`cleanTmpProject(tmpDir)` - recursively removes temp directory. Use `fs.rmSync(tmpDir, { recursive: true, force: true })`.

`runHook(event, stdinJson, projectDir)` - executes the hook via `execFileSync('node', [obsScript, event], { input: JSON.stringify(stdinJson), env: { ...process.env, CLAUDE_PROJECT_DIR: projectDir }, stdio: ['pipe', 'pipe', 'pipe'], timeout: 10000 })`. Returns void (throws on non-zero exit).

`readJsonl(filePath)` - reads file, splits by newline, filters empty, maps JSON.parse. Returns array of objects.

`getObsFile(tmpDir, project)` - returns `path.join(tmpDir, '.claude', 'logs', 'observations', project, 'obs.jsonl')`.
