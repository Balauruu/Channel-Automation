# Writer Memory

## Key Files
- Script output: projects/*/script/Script.md
- Script outline: projects/*/script/outline.md
- Revision notes: projects/*/script/revisions/
- Voice profile: channel/voice-profile.md
- Channel DNA: channel/channel.md
- Writer CLI: editorial/writer/cli.py (load, generate, revise)

## Decisions
- [2026-04-10] Scripts target 3,000-7,000 words (20-50 min at 150 words/min)
- [2026-04-10] 4-7 acts per script, each with dramatic question and transition hook
- [2026-04-10] Chapter titles use evocative register, not descriptive: "Strangers in the Jungle" not "Two Outsiders Arrive"
- [2026-04-10] Hook formula: opening quote -> compressed overview -> misinformation flag (if applicable) -> "This is the true story of..."
- [2026-04-10] No hedging ("reportedly", "allegedly") when claim is sourced -- reserve for genuinely speculative claims
- [2026-04-10] HOOKs are structural anchors (where chapters begin), QUOTEs are verbatim speech -- they are not interchangeable
- [2026-04-10] Open ending template used only for unsolved cases or contested resolutions -- never for cases with clear factual resolution

## Patterns
- [2026-04-10] Short declarative sentences (under 10 words) serve as structural breathing points after heavy information
- [2026-04-10] Third-person narrator, never first person -- narrator is invisible
- [2026-04-10] Specificity is credibility: names, dates, places -- never "a small town" when you know the town name
- [2026-04-10] Unsourced claims from dossier should not be included -- flag back to researcher
- [2026-04-10] Act transitions end with implication, not summary
- [2026-04-10] AI puffery ban: avoid pivotal, crucial, vital, testament, delve, leverage, multifaceted, foster, realm, tapestry
- [2026-04-10] QUOTEs always introduced by narrator with attribution pattern before the quoted text -- never absorbed into surrounding narration
